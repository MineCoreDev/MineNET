# MineNET
MineCraftBE Server SoftWere  
1から作ったサーバーソフトウェア

## 目標
とりあえず、最低限鯖を運営できる程度にする

## 現状
地形が保存されるようになり、IDが256以上のブロックが読み込まれるようになった

## MineNETで実装予定の要素
- Entityの実装
- レッドストーンの実装
など
